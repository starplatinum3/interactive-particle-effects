# interactive-particle-effects

一个以「星辰漫游」为主题的交互式粒子场景。直接打开 `index.html` 即可体验，包含密度调节、流星开关以及夜空/黎明主题切换，并支持鼠标移动和点击产生星尘爆裂。

新增一款风格不同的「流光花园」光羽效果，放在单独的文件中。

新增文件夹 `lumen-bloom/`，提供「光羽绽放」场景：围绕核心盘旋的彩色光羽随指尖聚拢、点击绽放，支持颗粒数量、吸引力、尾迹长度和色盘切换。

新增文件夹 `prism-orbit/`，体验「光晕花园」：多层环形光羽环绕旋转，支持调节层数、流动与厚度，切换极光/晨曦/暮光色盘，带脉动、微光与点击涟漪互动。

新增文件夹 `sonic-ripples/`，体验「声波绽放」：上传音轨或启用麦克风，随音乐驱动光波、环形光带与涟漪拖尾，支持密度、扩散与配色循环。

新增文件夹 `tidal-veil/`，体验「镜潮之湾」：玻璃控件调节光羽数量、流速、涟漪跨度，切换潮汐/霓虹/暮光色盘，支持柔光与拖尾；鼠标牵引丝带，点击或空格键激起环形波纹。

新增文件夹 `halo-drift/`，体验「光幕浮潮」：调节粒子数量、流速、旋涡力度与尾迹长度，切换多色光羽；柔光与拖尾自由开关，点击/空格激发光波，双击快速换色。

新增文件夹 `lunar-waves/`，体验「月光织潮」：玻璃控件调节粒子织线、潮汐流速与扭曲幅度，切换三套冷暖渐变色盘，开启柔光与拖尾；鼠标牵引光丝，点击/空格激起环形波纹，双击快速换色。

新增文件夹 `glass-spires/`，体验「Prism Drift」：碎片化的玻璃光羽随指尖流动，支持数量、流速、扭曲与尾迹滑杆，柔光与碎光可选；点击或空格生成光波涟漪，随机闪烁快速换色。

新增文件夹 `lotus-echo/`，体验「灵波莲境」：漂浮的莲光受指尖牵引，支持数量、漂移、绽放与涡流调节；可切换三套色盘，开启柔光、拖尾与心跳脉动，点击或空格激起环形涟漪。

新增文件夹 `stellar-cascade/`，体验「星弧流霞」：丝带光弧围绕中心分形游动，支持数量、流速与扭曲调节，可切换柔光、拖尾、呼吸脉动，色
盘双击切换；鼠标牵引光弧，点击或空格触发流动光涟。

## 使用
- 打开 `index.html`，使用顶部控制面板调整星尘密度与流星模式。
- 点击「切换到黎明/夜空」按钮改变配色与背景。
- 在画布上移动或点击，粒子会跟随或爆裂扩散。
- 打开 `aurora.html` 体验流光花园：拖动滑杆调整流动速度与光羽数量，点击切换色盘或关闭尾迹，鼠标移动牵引光带，点击生成柔和涟漪。
- 打开 `lumen-bloom/index.html` 体验光羽绽放：滑动控制颗粒数量、吸引力度和光羽长度，切换色盘或关闭闪烁，鼠标移动牵引光羽、点击制造绽放。
- 打开 `prism-orbit/index.html` 体验光晕花园：调整光羽层数、流动强度和厚度，切换色盘，开启/关闭脉动与微光，鼠标牵引环带、点击生成光波涟漪。
- 打开 `sonic-ripples/index.html` 体验声波绽放：导入音频或开启麦克风，播放/暂停音乐时光环与涟漪随能量律动，可调节涟漪密度、扩散幅度、配色循环与柔光拖尾。
- 打开 `halo-drift/index.html` 体验光幕浮潮：滑动粒子数量、流速、旋涡与尾迹长度，切换色盘，开启柔光/拖尾，鼠标牵引旋涡，点击或空格触发光波，双击更换配色。
- 打开 `lunar-waves/index.html` 体验月光织潮：调节粒子织线、潮汐流速与织纹扭曲，切换三套色盘，开启柔光与拖尾，鼠标牵引丝带，点击或空格掀起环形波纹，双击切换色盘。
- 打开 `glass-spires/index.html` 体验 Prism Drift：调节碎片数量、流速、扭曲与尾迹，切换色盘与柔光/碎光，鼠标牵引碎片，点击或空格激起光波，随机闪烁按钮快速换色。
- 打开 `lotus-echo/index.html` 体验灵波莲境：滑动控制莲光数量、漂移、绽放与涡流强度，切换雾光/暮耀/珊瑚色盘，开启柔光、拖尾与心跳脉动；移动指针牵引莲光，点击或空格生成环形涟漪。
- 打开 `stellar-cascade/index.html` 体验星弧流霞：调节光弧数量、流动速度与扭曲幅度，切换柔光/拖尾/呼吸，点击或空格触发光涟，双击快速换色。
