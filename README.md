# interactive-particle-effects

一个以「星辰漫游」为主题的交互式粒子场景。直接打开 `index.html` 即可体验，包含密度调节、流星开关以及夜空/黎明主题切换，并支持鼠标移动和点击产生星尘爆裂。

新增一款风格不同的「流光花园」光羽效果，放在单独的文件中。

新增文件夹 `lumen-bloom/`，提供「光羽绽放」场景：围绕核心盘旋的彩色光羽随指尖聚拢、点击绽放，支持颗粒数量、吸引力、尾迹长度和色盘切换。

## 使用
- 打开 `index.html`，使用顶部控制面板调整星尘密度与流星模式。
- 点击「切换到黎明/夜空」按钮改变配色与背景。
- 在画布上移动或点击，粒子会跟随或爆裂扩散。
- 打开 `aurora.html` 体验流光花园：拖动滑杆调整流动速度与光羽数量，点击切换色盘或关闭尾迹，鼠标移动牵引光带，点击生成柔和涟漪。
- 打开 `lumen-bloom/index.html` 体验光羽绽放：滑动控制颗粒数量、吸引力度和光羽长度，切换色盘或关闭闪烁，鼠标移动牵引光羽、点击制造绽放。
